Recuperación de Contraseña

Descripción General

El sistema proporciona una funcionalidad segura para que los alumnos puedan recuperar su contraseña mediante el correo electrónico registrado en la plataforma.

Proceso de Recuperación

  • Acceder a la página de recuperación de contraseña
  • Ingresar el correo electrónico registrado
  • Recibir el correo de recuperación
  • Seguir el enlace de restablecimiento
  • Crear nueva contraseña

Especificaciones Técnicas

Componente Descripción
Actor Principal Alumno
Precondiciones Usuario registrado en el sistema
Postcondiciones Contraseña recuperada exitosamente

Manejo de Excepciones

  • Validación de formato de correo electrónico
  • Verificación de existencia del usuario
  • Control de intentos fallidos
  • Tiempo límite de enlace de recuperación
