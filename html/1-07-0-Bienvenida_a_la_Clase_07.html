Ingeniería de Requerimientos

Explorando los Casos de Uso

Introducción

Bienvenidos a la materia de Ingeniería de Requerimientos. En este curso, exploraremos los casos de uso como herramienta fundamental para la documentación y diseño de sistemas.

Beneficios Principales

  • Facilita la comunicación entre desarrolladores y clientes
  • Define claramente las interacciones usuario-sistema
  • Documenta requisitos funcionales
  • Sirve como base para pruebas del sistema

Elementos Clave

Los casos de uso son fundamentales para:

  • Capturar requisitos funcionales
  • Modelar interacciones
  • Establecer alcance del sistema
  • Guiar el desarrollo
