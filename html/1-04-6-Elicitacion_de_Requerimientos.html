Elicitación de Requerimientos

¿Qué es la Elicitación?

La elicitación es el proceso sistemático de indagar, investigar y comprender una situación que necesita ser solventada. Este proceso es fundamental para el desarrollo de sistemas exitosos.

Preguntas Clave:

  • ¿Cómo es el sistema actual?
  • ¿Cuáles son sus problemas?
  • ¿Qué mejoras son posibles?

Técnicas de Recolección

  1. Brainstorming
  2. Cuestionarios
  3. Entrevistas
  4. Observación
  5. Análisis de documentación

Importancia del Dominio

Comprender el dominio del problema es esencial para proponer una solución adecuada. Este conocimiento permite:

  • Identificar necesidades reales del cliente
  • Desarrollar sistemas que satisfagan requerimientos
  • Optimizar procesos existentes
  • Proponer mejoras significativas

Fuentes de Requerimientos

Fuente Descripción
Stakeholders Personas interesadas o afectadas por el sistema
Usuarios Personas que utilizarán directamente el sistema
Expertos del dominio Especialistas en el área de negocio
Objetivos del negocio Metas y estrategias organizacionales
Reguladores Normas y estándares aplicables
