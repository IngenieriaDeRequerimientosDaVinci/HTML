Diagramas de Casos de Uso

Conceptos Fundamentales

Los diagramas de casos de uso son herramientas visuales esenciales que representan:

  • Interacciones entre usuarios y sistema
  • Funcionalidades principales del sistema
  • Relaciones entre diferentes componentes
  • Actores primarios y secundarios
Elemento Descripción
Actor Primario Inicia la interacción con el sistema
Actor Secundario Responde a las solicitudes del sistema
Caso de Uso Representa una funcionalidad específica
