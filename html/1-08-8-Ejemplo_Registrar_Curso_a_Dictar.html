Caso de Uso: Registrar Curso a Dictar

Este caso de uso permite a los profesores registrarse en los cursos que van a dictar durante el período académico.

Precondiciones

  • El profesor debe estar registrado en el sistema
  • Debe existir al menos un curso disponible
  • Se debe haber ejecutado el subflujo "Agregar Curso"

Flujo Principal

  1. El profesor inicia sesión en el sistema
  2. El sistema muestra las opciones disponibles
  3. El profesor selecciona "Registrar Curso a Dictar"
  4. El sistema muestra los cursos disponibles
  5. El profesor selecciona el curso deseado
  6. El sistema confirma el registro

Subflujos

Subflujo Descripción
Agregar Curso Permite añadir un nuevo curso al sistema
Modificar Curso Permite modificar los detalles de un curso existente
Eliminar Curso Permite eliminar un curso del sistema

Excepciones

  • Login inválido: El sistema muestra un mensaje de error
  • Curso no disponible: El sistema notifica que el curso no está disponible
  • Error de conexión: El sistema muestra un mensaje de error de conexión

Postcondiciones

  • El profesor queda registrado en el curso seleccionado
  • El sistema actualiza la lista de cursos disponibles
  • Se genera un registro de la asignación en el sistema
