Técnicas de Relevamiento de Datos

Temas Principales:

  • Relevamiento de datos
  • Fuentes de datos
  • Tipos de pregunta en cuestionarios
  • Etapas de una entrevista
  • Observación personal
  • Medición de tiempos

Técnicas de Elicitación

Técnica Descripción
Cuestionarios Herramienta estructurada para recopilar información de múltiples fuentes
Entrevistas Interacción directa para obtener información detallada
Observación Análisis directo de procesos y comportamientos

Estas técnicas son fundamentales para la elicitación de requerimientos, una fase crucial en el desarrollo de software. La efectiva aplicación de estas metodologías puede determinar el éxito del proyecto.
