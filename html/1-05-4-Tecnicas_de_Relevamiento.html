Técnicas de Relevamiento de Datos

Análisis comparativo de metodologías para la recolección efectiva de información

Cuestionario

Ideal para recolección masiva de datos

  • Permite alcanzar grandes poblaciones
  • Facilita el análisis estadístico
  • Bajo costo por respuesta

Entrevista

Óptima para datos selectivos y detallados

  • Permite profundizar en las respuestas
  • Facilita la aclaración de dudas
  • Mayor riqueza de información

Observación Personal

Recolección directa y objetiva de datos

  • Datos de primera mano
  • Minimiza sesgos de interpretación
  • Captura comportamientos reales

Medición de Tiempos

Análisis de eficiencia de procesos

  • Optimización de procedimientos
  • Identificación de cuellos de botella
  • Mejora de productividad
Técnica Ventajas Desventajas Uso Recomendado
Cuestionario Alcance masivo, bajo costo Respuestas superficiales Estudios estadísticos
Entrevista Información detallada Mayor tiempo y costo Análisis en profundidad
Observación Datos objetivos Requiere presencia física Análisis de comportamiento
Medición Datos precisos Limitado a procesos medibles Optimización de procesos
